# Fiind Iframe Preview

This repository hosts a single-page preview for the Fiind booking embed widget.
It is intended for quick environment and viewport testing before production use.

## Files

- `index.html`: GitHub Pages entry file.
- `iframe.html`: Source page used for local editing/testing.

## Local Preview

Open `index.html` in your browser, or run a simple local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## GitHub Pages

After enabling Pages in the repository settings, the site will be available at:

`https://mukund2074.github.io/fiind-iframe/`
